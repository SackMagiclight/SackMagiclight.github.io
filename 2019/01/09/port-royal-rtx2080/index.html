# 構成

  • CPU : Ryzen7 2700X stock
  • GPU : EVGA GeForce RTX 2080 XC ULTRA GAMING
  • M/B : Asrock X370 Gaming K4

# 結果

Score : 5921

平均FPSは30ぐらい。
RTX2080Tiだと8200、RTX2070だと4900程のようです。
RTX2080のコスパが良くない気がする
